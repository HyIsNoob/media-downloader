# Instructions for yt-dlp Installation

To use this app, you need to install yt-dlp on your system. Here's how:

## Windows

1. Download the latest yt-dlp.exe from https://github.com/yt-dlp/yt-dlp/releases
2. Place the downloaded exe file in this directory
3. Alternatively, you can add yt-dlp to your system PATH

## macOS

1. Install via Homebrew:
   ```
   brew install yt-dlp
   ```
2. Or download the binary and place it in this directory

## Linux

1. Install via your package manager:
   ```
   # Arch Linux
   sudo pacman -S yt-dlp
   
   # Debian/Ubuntu
   sudo apt install python3-pip
   sudo pip install yt-dlp
   ```
2. Or download the binary and place it in this directory

Note: For audio conversion features, you should also install ffmpeg on your system.
